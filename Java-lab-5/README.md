# Лабораторная работа №5
Тема: Использование циклов в языке программирования Java

## Описание

В данной лабораторной работе реализованы программы с использованием
циклов while, do-while и for.

## Список файлов

Task1.java — Вывод чисел от 1 до N (цикл for)
Task2.java — Сумма чисел от 1 до N (цикл while)
Task3.java — Вычисление факториала (цикл for)
Task4.java — Вывод четных чисел от 1 до 100 (while + continue)
Task5.java — Ввод чисел до 0 и вычисление суммы (do-while)
Task6.java — Таблица умножения 1–5 (вложенные циклы)
