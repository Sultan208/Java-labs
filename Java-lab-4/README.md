Лабораторная работа №4
Работа с условными операторами и операторами выбора в языке Java.

Цель работы — изучение условных операторов if, else, switch и закрепление навыков обработки пользовательского ввода.

В лабораторной работе реализованы следующие программы:

SumExample.java — вычисление суммы чисел от 1 до введённого числа N.

EvenNumbers.java — вывод чётных чисел в диапазоне от 1 до 20.

FactorialExample.java — вычисление факториала введённого числа.

OddNumbers.java — вывод нечётных чисел от 1 до 100 с использованием цикла do-while.

SumUntilZero.java — ввод чисел пользователем и вычисление их суммы до ввода числа 0.

PositiveNegativeCheck.java — определение, является ли число положительным, отрицательным или равным нулю.

GradeCalculator.java — определение оценки студента по пятибалльной шкале на основе введённого балла.

CompareNumbers.java — сравнение двух введённых чисел.

DayOfWeek.java — определение дня недели по его номеру.

EvenOddCheck.java — проверка числа на чётность.

Для запуска программ необходимо открыть терминал в Visual Studio Code и выполнить команды:

javac НазваниеФайла.java
java НазваниеФайла

Пример запуска программы:

javac SumExample.java
java SumExample

Используемые технологии: язык программирования Java, среда разработки Visual Studio Code.
