# Лабораторная работа №2 (Карали Султан)

## Тема
Переменные, типы данных и наследование в Java

## Цель
Закрепить навыки работы с переменными, типами данных и механизмом наследования.

## Содержание
- Переменные и константы
- Примитивные и ссылочные типы
- Область видимости переменных
- Наследование классов
- Переопределение методов
- Полиморфизм

## Структура проекта
- Person.java — базовый класс
- Student.java — производный класс
- Main.java — демонстрация наследования
- VariablesDemo.java — переменные и константы
- TypesDemo.java — типы данных
- ScopeDemo.java — область видимости
