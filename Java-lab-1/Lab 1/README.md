# Лабораторная работа №1

## Тема
Введение в язык программирования Java. Работа в среде разработки IntelliJ IDEA.

## Цель работы
Ознакомиться с основами языка Java и структурой простой Java-программы.

## Описание
Программа выводит информацию о студенте (ФИО, группа, специальность) в консоль.

## Как запустить
1. Открыть проект в IntelliJ IDEA
2. Запустить файл `StudentInfo.java`
