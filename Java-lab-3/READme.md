Описание программы:
Программа содержит решения заданий лабораторной работы №3.
Реализованы линейные алгоритмы без использования условий и циклов.

Список файлов программы:

1. SumThreeNumbers.java
Программа складывает три целых числа и выводит результат.

2. ArithmeticOperations.java
Программа выполняет сложение, умножение и деление двух чисел, введенных пользователем.

3. AssignmentExample.java
Программа демонстрирует изменение значения переменной с использованием оператора присваивания.

4. NumberSquare.java
Программа запрашивает число у пользователя и вычисляет его квадрат.

5. RectangleArea.java
Программа вычисляет площадь прямоугольника по введенным длине и ширине.

Инструкция по запуску:
1. Открыть проект в Visual Studio Code.
2. Запускать каждый файл отдельно.
3. Следовать инструкциям в консоли.

Используемые технологии:
Язык программирования Java
Среда разработки Visual Studio Code
