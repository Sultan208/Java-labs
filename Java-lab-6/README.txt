Лабораторная работа №6
Тема: Методы, параметры методов и рекурсивные функции в Java
Описание работы:

В данной лабораторной работе реализованы методы с параметрами и возвращаемыми значениями.
Также продемонстрирована перегрузка методов и использование рекурсии.

Список файлов:

1. Task1_RectangleArea.java
   - Метод rectangleArea(double width, double height)
   - Вычисляет площадь прямоугольника.

2. Task2_IsEven.java
   - Метод isEven(int number)
   - Определяет, является ли число чётным.

3. Task3_MaxMethod.java
   - Перегруженные методы max(int a, int b) и max(double a, double b)
   - Возвращают большее из двух чисел.

4. Task4_Factorial.java
   - Рекурсивный метод factorial(int n)
   - Вычисляет факториал числа.
   - Содержит базовый случай и проверку на отрицательное число.

5. Task5_Power.java
   - Рекурсивный метод power(int base, int exponent)
   - Вычисляет степень числа.
   - Использует базовый случай exponent == 0.
